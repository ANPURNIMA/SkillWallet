# **App Name**: VidyaMitra

## Core Features:

- User Authentication & Profile: Secure user registration and login. Display user-specific profile details including 'Credits' and 'AI Tokens' counters.
- Skill Bank & Wallet Management: Allows users to add, view, and manage their skills within a personal 'Skill Bank' or 'Skill Wallet' section.
- AI-Powered Resume Evaluator Tool: Leverages AI to analyze user resumes, provide evaluation feedback, and map identified skills to career paths.
- AI-Driven Mock Interview Tool: Facilitates AI-powered mock interviews, providing real-time feedback and improvement suggestions to users.
- AI-Powered Career Planner Tool: Generates personalized career path recommendations and suggests relevant upskilling opportunities based on user data.
- Credit and AI Token Tracking: Manages and updates user 'Credits' and 'AI Tokens' for accessing various platform features.
- Landing Page Content Display: Displays static and dynamic content for the landing page, including project overview, platform description, and project progress.

## Style Guidelines:

- Primary color: Deep blue (#1e3a8a). This robust blue evokes professionalism and intelligence, aligning with the educational and agent themes. It provides strong contrast against the light background.
- Background color: Very light blue-gray (#FAF9FB). This almost-white, heavily desaturated shade of blue maintains a clean aesthetic while subtly tying into the primary color palette.
- Accent color: Cool slate-gray (#7B858F). This muted, desaturated cool-gray provides a sophisticated counterpoint, adding depth and professional elegance while remaining harmonious with the primary blue.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, techy, and scientific feel, ideal for showcasing an 'Intelligent Career Agent'.
- Body font: 'Inter' (sans-serif) for its objective, neutral, and highly readable quality, ensuring clarity for platform descriptions and feedback.
- Utilize sleek, modern, and conceptually relevant line icons that complement the Ed-Tech aesthetic without appearing overly playful. Focus on clarity and intuitive representation of features.
- Adopt a clean, spacious layout with a dominant central content area and a supportive right sidebar. Ensure full responsiveness, optimizing for both mobile and desktop experiences, maintaining readability and navigation ease.
- Incorporate subtle, smooth transitions and micro-interactions using Framer Motion, particularly for scenario card changes and accordion menu expansions, to enhance user engagement and provide a fluid experience.
